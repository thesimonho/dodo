## What it does

You're working on a project. Your documentation is never up to date. Different people need different things from your docs, and keeping all of them fresh is a chore nobody wants to do.

dodo looks at what documentation you have (or don't have), and either updates it or offers to create it.

```
/dodo
```

That's it.
